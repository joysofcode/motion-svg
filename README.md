## Local Development

### 🧑‍🤝‍🧑 Clone the project

```sh
https://github.com/joysofcode/motion-svg.git
```

### 📦️ Install dependencies

```sh
pnpm i
```

### 💿️ Run the development server

```sh
pnpm run dev
```
